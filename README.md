# My Drink

Uma vitrine interativa de bebidas em uma única página, construída com HTML, CSS e JavaScript.

## Visão Geral

Este projeto exibe um carrossel de imagens de frutas com tema de bebida, com título em destaque e transições animadas. Os usuários podem navegar entre os slides usando os botões anterior e próximo.

## Funcionalidades

- Apresentação de slides de bebidas com frutas
- Animação suave de entrada para imagem e título
- Navegação manual com botões anterior/próximo
- Layout simples e responsivo com propriedades CSS

## Arquivos

- `index.html` - marcação principal da página
- `style.css` - estilos para o layout do carrossel, botões e animações
- `scripts.js` - lógica de navegação dos slides
- `img/` - imagens usadas em cada slide de bebida

## Uso

1. Abra `index.html` em um navegador.
2. Clique nos botões `<` ou `>` para alternar entre os slides de bebidas.

## Observações

- O carrossel foi criado sem bibliotecas externas.
- Usa JavaScript puro para manipulação do DOM e eventos.
- O slide ativo inicial é o primeiro item (`Morango`).
